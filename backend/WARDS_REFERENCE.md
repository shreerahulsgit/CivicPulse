# Chennai Corporation (GCC) — Zones & Wards Reference

> **15 Zones · 200 Wards · Greater Chennai Corporation**
> Use this file when creating Zonal Officers and Ward Officers via the Admin panel.

---

## How to use

| Field | Value |
|-------|-------|
| zone_id | DB primary key (equals zone_number, 1–15) |
| ward_number | Official GCC ward number (1–200) |

When creating a **Zonal Officer** in the Admin panel, set zone_id = the zone number below.
The Ward Officer is created by the Zonal Officer themselves from their dashboard.
